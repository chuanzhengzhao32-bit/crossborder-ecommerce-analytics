# 指标字典

| 指标 | 公式 | 分析用途 | 注意事项 |
|---|---|---|---|
| 有效收入 | 订单收入 - 退款金额 | 衡量最终保留收入 | 统一币种和退款确认时间 |
| 有效订单数 | 非取消且未全额退款的订单数 | 衡量订单质量 | 明确部分退款是否保留 |
| 商品成本 | 销售数量 × SKU 单位采购成本 | 计算毛利 | 成本需按销售时点匹配 |
| 毛利 | 有效收入 - 商品成本 | 判断商品基础盈利能力 | 尚未扣除履约和获客费用 |
| 毛利率 | 毛利 / 有效收入 | 横向比较商品盈利结构 | 收入为 0 时返回空值 |
| 贡献利润 | 毛利 - 平台费 - 支付费 - 物流费 - 广告费 | 判断是否值得继续投入 | 广告归因需统一 |
| 贡献利润率 | 贡献利润 / 有效收入 | 比较店铺/SKU真实经营质量 | 结合规模共同判断 |
| ROAS | 广告归因收入 / 广告消耗 | 判断广告收入效率 | 不等同于利润 |
| 有效订单成本 | 广告消耗 / 广告归因有效订单数 | 衡量获客成本 | 排除退款订单 |
| 库存金额 | 可售库存 × 单位库存成本 | 判断资金占用 | 需区分可售、残次和在途 |
| 库存周转天数 | 平均库存成本 / 销售成本 × 天数 | 判断库存效率 | 与缺货率联合解释 |
| 缺货率 | 缺货 SKU 数 / 在售 SKU 数 | 判断供给风险 | 重点 SKU 可加权 |
| 滞销库存金额 | 滞销 SKU 可售库存 × 单位库存成本 | 判断清仓压力 | 滞销天数按品类设定 |
